# I2C Sensors

Project to toy around with Circuit Python on RaspberryPi using Adafruit Blinka translation layer.

Sensor:
 - Adafruit BME680

## Getting started

```
python3 -m venv .venv
. ./.venv/bin/activate
pip install --upgrade pip
pip install -r requirements.in

# Setup as a Python module
python -m sense
```
